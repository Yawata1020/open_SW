2. 문서 편집기
- vi, vim, emacs, notepad++, ...
- "vi"sual editor
  - vim - vi improved (정규표현식, ...)
- 프롬프트에서 파일을 직접 편집할 수 있게 해주는 툴
- 특징
  - 화면 단위 편집기
  - 유닉스 계열에서 가장 많이 사용됨
  - 텍스트 편집 기능만 제공함
  - "less"
  - 모드: 일반모드 <-> 입력모드 <-> 명령모드
    - ESC로 모드 전환
    - 명령모드 : 파일 저장, 덮어쓸건지, 빠져나갈건지 등
    => vi로 파일 열기 -> 입력모드 -> ESC -> 빠져나가기

    - 입력모드 :
      - i: 현 커서에서 입력
      - a: 현 커서 한 칸 뒤에서 입력
      - o: 한 줄 추가 후 입력
    - 명령모드
      :wq! + enter

- 파일 아무거나 열어서 수정하고 저장하고 나오기
